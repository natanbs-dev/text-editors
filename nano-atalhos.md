## fora os básicos, segue a lista

- **CTRL + K** = recortar o conteúdo
- **CTRL + U** = cola o que foi copiado
- **CTRL + C** = consegue verificar quantos caracteres foram inseridos até aquela linha, quantas linhas tem o arquivo
- **CTRL + A**  = vai para o inicio da linha
- **CTRL + E** = vai para o final da linha
- **ALT + A** = seleção de texto mais complexo
- **ALT + 6 (M-6)** = copia a linha
- **Alt + U** =  Desfazer (undo)
- **Alt + N** = Ativar/desativar numeração de linhas
